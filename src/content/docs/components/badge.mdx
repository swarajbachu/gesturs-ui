## Usage

The `Badge` component is a container that displays a badge with a text and color.

## Props

### Variants

The `Badge` component 2 variants: `filled` and `light`.

default variant is `filled`

<ComponentPreview name="/ui/badge-demos/badge-variants" />

### Color

The `Badge` component has multiple color variants: `primary`, `secondary`, `destructive`, and `accent`.

also along with them you can use your regular tailwind colors like `red`, `blue`, `green` etc.

default color is `primary`

<ComponentPreview name="/ui/badge-demos/badge-colors" />

### Size

The `Badge` component has 3 sizes: `sm`, `md`, and `lg`.

default size is `md`

<ComponentPreview name="/ui/badge-demos/badge-sizes" />

### Sections

The `Badge` component has 2 sections: `leftSection` and `rightSection`.

<ComponentPreview name="/ui/badge-demos/badge-sections" />

### Props

| Name           | Type        | Default   | Description                     |
| -------------- | ----------- | --------- | ------------------------------- |
| `variant`      | `string`    | `filled`  | The variant of the badge.       |
| `color`        | `string`    | `primary` | The color of the badge.         |
| `size`         | `string`    | `md`      | The size of the badge.          |
| `leftSection`  | `ReactNode` | `null`    | The left section of the badge.  |
| `rightSection` | `ReactNode` | `null`    | The right section of the badge. |

rest of the props are passed to the `span` element
