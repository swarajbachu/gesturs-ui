## Usage

```tsx
<GooeyMenu>
  <GooeyMenuBefore>
    <FaLinkedin />
    <FaInstagram />
  </GooeyMenuBefore>
  <GooeyMenuTrigger>
    <FaShare />
  </GooeyMenuTrigger>
  <GooeyMenuAfter>
    <FaFacebook />
    <FaTwitter />
  </GooeyMenuAfter>
</GooeyMenu>
```

## Props

### Direction

direction can be `vertical` or `horizontal` and it will decide the direction of the menu. default is `horizontal`.

make sure to add `id` to `GooeyTrigger` if you are using multiple `GooeyMenu` in the same page.
